Implementar una página HTML estática en Heroku

basado en: https://medium.com/@agavitalis/how-to-deploy-a-simple-static-html-website-on-heroku-492697238e48


Heroku es un de servicios en la nube.

Proporciona alojamiento para apps creadas con tecnologías como:

  • Node.js
  • Ruby
  • Java
  • PHP
  • Python
  • Go
  • Scala
  • Clojure

Requisitos para implementar una aplicación en Heroku

  • Una cuenta de Github
  • Una cuenta de Heroku
  • Git instalado en computador
  • CLI de Heroku instalada en computador

Heroku solo aloja y admite aplicaciones y no páginas estáticas, para implementar nuestra página html estática haremos que Heroku interprete nuestro sitio web como una aplicación PHP.

Pasos

1) Crear un archivo index.php con: <?php include_once(“home.html”); ?>

2) Crear un archivo home.html con: <h1>hola heroku</h1>

3) Crear un archivo composer.json con {}para que se reconozca como una aplicación PHP

En la terminal o bash:

4) git init iniciar git

5) git add . agregar cambios

6) git commit -m "desplegando" comitear cambios

7) Heroku login autetificarse en Heroku

8) heroku apps:create NOMBRE_DE_APP crear app en Heroku

9) git push heroku master subir cambios a Heroku

10) Aquí está el enlace de tu app: https://NOMBRE_DE_APP.herokuapp.com

11) git add . git commit -m "mensaje sobre cambios" git push heroku master actualizar
